# Contributing to Nodeterm SSH MCP Server

Thank you for your interest in contributing! We welcome all contributions that help improve this MCP server.

## How to Contribute

1.  **Fork the Repository**: Create your own copy of the project.
2.  **Create a Branch**: Use a descriptive name for your feature or fix (e.g., `fix/connection-timeout`).
3.  **Make Your Changes**: Ensure your code follows the existing style and is well-documented.
4.  **Test Your Changes**: Verify that the MCP server works as expected with Gemini CLI or Claude Desktop.
5.  **Submit a Pull Request**: Provide a clear description of what your changes do.

## Development Setup

1.  Clone your fork.
2.  Run `npm install` to install dependencies.
3.  Test locally using `node index.js`.

## Code of Conduct

Please be respectful and professional in all your interactions within this project.
